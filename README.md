# POS Radar

Windows diagnostic and recovery tool for POSiFLORA, ATOL/UEM services, cash register connectivity, USB/COM issues, and OFD troubleshooting.

The current repository contains:

- `uem_monitor.py` — legacy Python/PyInstaller prototype for UEM monitoring.
- `docs/superpowers/specs/2026-04-27-posiflora-recovery-tool-design.md` — approved MVP design.
- `docs/superpowers/plans/2026-04-28-posiflora-recovery-tool.md` — implementation plan.

Target stack:

- .NET 10 LTS
- Windows Worker Service
- WPF
- Wpf.Ui
- xUnit
